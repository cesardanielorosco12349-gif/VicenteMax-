# Instrucciones para actualizar el README

1. **Abre el repositorio en GitHub.**
2. **Haz clic en el botón "Code" y luego en "README.md".**
3. **Haz clic en el lápiz para editar el archivo.**
4. **Escribe `/update` y presiona Enter para que Copilot te ayude.**
5. **O directamente pega el texto que te di anteriormente.**
6. **Haz clic en "Commit changes" para guardar los cambios.**

**O con comandos de Git:**

1. **Abre la terminal en tu computadora.**
2. **Escribe `git pull` para actualizar el repositorio local.**
3. **Escribe `nano README.md` para editar el archivo (en Linux/macOS) o `notepad README.md` (en Windows).**
4. **Pega el texto que te di anteriormente.**
5. **Escribe `git add README.md` para agregar los cambios.**
6. **Escribe `git commit -m "Actualización de README.md"` para guardar los cambios.**
7. **Escribe `git push` para enviar los cambios al repositorio remoto.**
